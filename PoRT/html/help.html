Help guide

This guide will help you understand how our site works in order to clarify any concerns or questions you may have.

Nowadays there are numerous activities with negative impact on the environment. Our goal is to keep track of these unpleasant events so that they can be verified and, eventually, resolved by environmental inspectors. This platform offers users the opportunity to complete a form for reporting these kind of activities.

What do the menu buttons do?

By pressing this button, you return to the main page.
By pressing this button, you can view the map and places where all the reported disasters occurred.
By clicking this button, you can view the list of all the disasters that have been reported so far.
This button makes it possible for you to report and add a new disaster to the current list.
By clicking this button, you will be redirected to the help page.
